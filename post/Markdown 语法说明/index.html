文档参数

---
# 文章标题, (可选,默认为 文件名)
title: 标题
# 博客标签, 字符串数组,  (可选,默认为 空)
tags: [标签 1, 标签 2]
# 发布日期, 格式为YYYY-MM-DD, (可选,默认为 空)
date: 2000-01-01
# 是否加密, (可选,默认为false)
encrypt: true
# 是否草稿, 如果为 true, 文章将会渲染, 通过链接访会 404, (可选,默认为 false)
draft: true
# 是否是篇博客, 如果为 false 文件可以通过链接访问到但不会展示到页面 (可选,默认为 true)
post: false
---
