# nuxt-blog

一个基于 Nuxt 的极简博客系统, 只需添加 Markdown 文件即可轻松创建美观的文档站点。

## 运行

```shell
# 安装依赖
pnpm install

# 开发环境
pnpm dev

# 打包静态资源
pnpm generate
```

## 特征

+ 无需服务器静态 HTML 打包支持
+ 无需配置路由, 只需 Markdown 即可轻松创建美观的文档站点。
+ 博客加密功能
